核心特性

📚

海量词库

136,000 词条内置词典（jieba）+ 1,280,000 条 2-gram 语言模型（Wikipedia 训练），覆盖日常输入 99%+ 场景

🧠

智能整句切分

DP 动态规划 + 2-gram 重排，输入 "woaixuexi" 直接识别为 "我爱学习"，支持 3-gram 上下文联想

🎯

强大容错

模糊音（z/zh, n/l, f/h 等方言）+ 邻键容错 + 声母遗漏 + 拼写纠错，输入 "nigao" 也能出 "你好"

💾

用户词典持久化

BadgerDB 嵌入式存储，跨进程保留用户习惯，时间衰减让最近输入权重更高

🔗

上下文联想

2-gram + 3-gram 共现模型，提交"我"后输入"n"优先返回"你"，自动学习用户输入模式

⚡

硬件加速渲染

Windows Direct2D + macOS NSVisualEffectView 毛玻璃，GPU 加速候选词窗口，触摸手势翻页

🔤

单字母联想

输入 "n" 返回高频字（你/年/那/能），输入 "nh" 联想 "你好"，首字母缩写快速输入

📋

剪切板历史

自动保存最近 50 条提交记录，支持搜索和恢复，N-gram 自动剪枝避免内存增长

🌍

跨平台原生

Windows TSF COM + macOS IMK + Linux IBus/Fcitx5，三平台原生适配，Go 核心引擎共享
